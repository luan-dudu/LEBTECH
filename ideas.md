# Brainstorming de Design - LEB TECH

## Abordagem Selecionada: Modernismo Corporativo com Gradientes Dinâmicos

### Design Movement
**Modernismo Corporativo Futurista** - Inspirado em interfaces de tecnologia de ponta, combinando geometria limpa com elementos dinâmicos que sugerem movimento e inovação.

### Core Principles
1. **Contraste Estratégico**: Fundo escuro (preto/azul profundo) com acentos vibrantes (azul brilhante, roxo)
2. **Hierarquia Clara**: Tipografia robusta e espaçamento generoso para guiar o olhar
3. **Movimento Sutil**: Transições suaves e animações que reforçam a sensação de tecnologia avançada
4. **Autenticidade Corporativa**: Design profissional que inspira confiança sem ser genérico

### Color Philosophy
- **Primário**: Azul vibrante (#0066FF) - representa confiança, tecnologia e inovação
- **Secundário**: Roxo (#7C3AED) - adiciona sofisticação e criatividade
- **Fundo**: Preto profundo (#0F0F0F) com gradientes sutis para profundidade
- **Acentos**: Branco e cinza claro para contraste legível
- **Intenção Emocional**: Profissionalismo + Inovação + Confiabilidade

### Layout Paradigm
- **Hero assimétrico**: Imagem/gradiente à direita, conteúdo à esquerda
- **Cards flutuantes**: Elementos com sombras suaves e bordas arredondadas moderadas
- **Seções alternadas**: Fundo claro/escuro para ritmo visual
- **Grid responsivo**: 1 coluna mobile, 2-3 colunas desktop com alinhamento dinâmico

### Signature Elements
1. **Gradiente diagonal**: Azul → Roxo usado em CTAs e destaques
2. **Linhas decorativas**: Linhas finas em azul como separadores e elementos de design
3. **Ícones geométricos**: Formas simples que representam cada serviço

### Interaction Philosophy
- Hover effects suaves com mudança de cor e elevação
- Botões com efeito de "ripple" ao clicar
- Transições de página suaves com fade-in
- Feedback visual claro em formulários

### Animation Guidelines
- Duração padrão: 300-400ms para transições
- Easing: ease-in-out para movimento natural
- Entrance: fade-in com slide leve (100px)
- Hover: escala 1.05 com mudança de cor
- Loading: spinner com gradiente animado

### Typography System
- **Display (Títulos principais)**: Poppins Bold 700 - 48px/56px
- **Heading (Subtítulos)**: Poppins SemiBold 600 - 28px/36px
- **Body (Texto)**: Inter Regular 400 - 16px/24px
- **Small (Labels)**: Inter Medium 500 - 14px/20px
- **Hierarquia**: Peso > Tamanho > Cor para diferenciar níveis

## Implementação
Este design será implementado com:
- Tailwind CSS 4 com tema customizado
- Componentes shadcn/ui adaptados
- Animações com Framer Motion
- Imagens geradas com IA para seções principais
- Responsividade mobile-first
